CtrlV now has a public API for uploading and manipulating images. You can use the Image Uploader to allow your website's users to upload images. Then you can use the REST API to manipulate those images as necessary.

Image Uploader

To use the CtrlV uploader include our javascript file on your page:

<script src="https://api.ctrlv.in/upload.js"></script>

Then call ctrlv.upload():

ctrlv.upload(function(image){
    // image is an object containing information about the uploaded image.
    console.log(image);
});

The contents of the image object are the same as those returned by Get An Image below. The object contains an additional property named key that can be used to manipulate the image in future requests.

A demonstation can be seen here.

Image Annotations

Add an Annotation

Add an annotation to an image. This will replace an existing annotation if it exists.

https://api.ctrlv.in/image/{id}/annotation

Parameter

Field Type Description Default Allowed Values
base64 required

string

Base64 encoded image to use as the annotation. This does not have to have the same dimensions as the image, but it must be the same ratio. It will be resized to the size of the image.
sessionKey required

string

Session key for the user that owns the image. Either sessionKey or imageKey is required.
imageKey required

string

Editing key for the image (obtained when the image is created). Either sessionKey or imageKey is required.
  • Success Response 
{
    "success": true,
    "image": [...] // See: Get An Image
}
